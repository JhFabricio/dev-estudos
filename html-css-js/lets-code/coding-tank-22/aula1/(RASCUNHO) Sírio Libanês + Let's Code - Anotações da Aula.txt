Para começar em programação, alguns conceitos precisam ficar claros e ter as suas definições bem explicadas para que os aprendizados posteriores façam sentido. O primeiro conceito que gostaria de definir é "programa", o que exatamente é um programa? Dentre as mais diversas definições e respostas, vamos trabalhar em cima da ideia de que programa é uma sequência lógica de instruções, nessa sequência está incluso o que você precisa para fazer, como fazer e como saber se está sendo feito corretamente, de fato uma sequência lógica de instruções. Essa definição se estende também para o conceito de "algoritmos", onde temos como analogias dois exemplos interessantes de serem citados, a preparação de um bolo e atravessar a rua, aliás, no segundo exemplo existe a presença de estruturas condicionais, conceito que será abordado em um outro tópico. Existem determinados jogos onde podemos aprender e exercitar a lógica de programação enquanto nos divertimos, eu recomendo o "Jogos do Blockly" e o "Scratch". Assim como utilizamos linguagens e idiomas para nos comunicarmos, como o português e o inglês, para nos comunicarmos com a máquina precisamos também aprender alguma linguagem que ela entenda, essas linguagens são chamadas de "linguagens de programação", uma muito usada e que acredito ser uma boa para começarmos é justamente o "JavaScript", mas vale ressaltar que ela não é a única, existem outras e todas com as suas individualidades e diferenças. 

Através das liguagens de programação conseguimos passar para o computador as instruções lógicas que queremos que ele execute, mas essas instruções precisam estar em alguma liguagem de programação para que ele entenda exatamente o que queremos dizer, é nesse ponto em que o JavaScript entra. Para programar em JS você precisa ter instalado alguma IDE, eu particularmente recomendo o Visual Studio Code, mas caso a sua máquina seja não seja tão potente o Sublime Text dá conta do recado. Aliás, existe uma lista de extensões para o VSCode que podem ser de grande ajuda para tornar mais fácil a tarefa de escrever códigos: "Pacote de Idioma Português" (Altera o idioma), "Auto rename tag" (Para alterar as tag), "Dracula Official" = Tema de estilização, "Live Server" (Auxilia nos testes), "VSCode-icons" (Coloca ícones nos arquivos) e, por fim, o "Code Runner" (Para rodar os seus códigos). Falando agora sobre uma parte mais prática, devemos levar em consideração na hora de programar que o computador apenas faz o que lhe é mandado, portanto, quem programa precisa ser o mais específico possível na hora de escrever a lista de instruções lógicas, para que o computador conclua com êxito o que lhe foi mandado. Vale lembrar que é importante escrever códigos legíveis, afinal, nós codificamos para humanos lerem, apesar de ser a máquina que vai executar, um humano irá ler o seu código e fazer as devidas correções. Já escolheu qual IDE usar? Estamos prontos então para começar a aprender alguns códigos e comandos.

Logo que você abre o seu navegador, clicando com o botão direito e depois na opção "Inspecionar" vai abrir a ferramenta de desenvolvedores do seu navegador, onde você pode ver os códigos que foram usados para a construção daquele site, há uma aba chamada "Console", vamos trabalhar bastante com ela. Gostaria de lembrar que esse resumo é especificamente dos assuntos abordados na primeira aula, logo, não vamos falar sobre todos os comandos possíveis, apenas sobre alguns que são importantes para começar a programar em JavaScript. Primeiramente, vamos entender o que são variáveis e porque elas vão ser importantes nesse início, variáveis alocam um espaço de memória para representar algum valor, elas são importantes para trabalharmos com valores, armazendo esses valores através das variáveis. As variáveis podem ser classificadas como "var", "const" e "let", sendo "var" a tradicional genética, "const" uma variável com valor imutável e "let" sendo uma variável com um valor flexível que pode ser alterado. Para criar uma variável é necessário decidir de que tipo ela será, vale lembrar que o JavaScript é uma linguagem que não é tipada, portanto não precisamos declarar as variáveis, mas não se apegue tanto à isso, é assunto para outra hora. Tendo decido se a sua variável vai ser uma var, const ou let você precisa nomeá-la, vale lembrar que o nome não começa com letra maiúscula, nem números e muito menos caracteres especiais, leve em consideração também o padrão da sua equipe, siga um padrão de escrita para que haja uma linguagem uniforme dentro do seu time. Tendo já sido decidido a variável e o nome, após o sinal de "=" para definir atribuição, vamos agora determinar de qual tipo será essa variável, temos como opções de variáveis de tipos primitivos: String (textos e alfanuméricos, representados sempre estando dentro dos sinais de " "), Number (números inteiros e fracionados), Boolean(Verdadeiro ou Falso), Undefined (Valor indefinido), como variáveis de tipos complexos temos: Function (funções) e Object (objetos). Por fim, determine qual valor será atribuído à variável e está feito, uma variável foi criada. Uma curiosidade: NaN é mostrado como number, mas significa "Not a Number", ele é um possível resultado caso haja erro de entendimento do tipo da variável por parte da máquina e ela acreditar que o resultado esperado fosse um número. 

Lembra que falamos sobre o Console do Inspecionar no navegador? Vamos falar mais dele agora e entender como ele funciona. Podemos usar um comando para exibir o que quisermos no Console, o comando utilizado para isso é o "console.log()", que de forma exemplificada funciona da seguinte maneira: digamos que queremos saber o resultado da soma "10 + 2" e queremos ver esse resultado no Console, para isso basta escrevermos no nosso arquivo .js(arquivo JavaScript) utilizando a IDE escolhida "console.log(10 + 2)", ao abrir o Console você verá nele o seguinte resultado "12", como desejado. Sabe aquelas janelas que se abrem quando você em determinados botões no site ou quando você simplesmente abre o site? São Janelas de alerta, para criar uma basta utilizar o comando "window.alert()" dentro dos () você escreve o que quer apresentar, se for uma variável apenas escreva seu nome, mas caso seja um texto não se esqueça que esse texto precisa estar entre "" ou ''. Como podemos ver, o computador entende comandos de aritmética e operações básicas sendo elas representadas pelos seus sinais tradicionais, para operações mais complexas como utilizar o valor de PI é necessário mexer com a biblioteca Math, mas não vamos nos aprofundar nisso nesse momento. Existem casos em que vamos querer alterar o tipo de determinada variável, transformando um "25" string em um "25" numérico, nesse caso exite um comando chamado "parseFloat() que nos possibilita fazer isso, usando o exemplo citado: temos uma variável "let numero1 = '25'" que queremos transformá-la de string em number, para isso deve escrever "numero2 = parseFloat(numero1)", teremos então alterado o tipo da nossa variável. Caso queira saber qual o tipo de determinada variável, usamos o comando "typeof", usando ainda a variável "numero2 = 25" podemos verificar se essa variável é mesmo numérica escrevendo "console.log(typeof numero2)" teremos o tipo da variável apresentado no Console, nesse caso o resultado é "number". Eu sei que falamos de forma bem corrida e superficial sobre esses conceitos tão abstrados, mas na medida que você for estudando e praticando vai fixar e entender de forma mais clara como programar em JavaScript funciona, o meu objetivo aqui hoje é apenas abordar de forma simples funcionalidades importantes do JavaScript para um iniciante e sobre como utilizá-las.
